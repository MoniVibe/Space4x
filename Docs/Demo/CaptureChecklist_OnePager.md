# Capture Checklist (One Pager)

## Pre-Capture
- Open render scene (`TRI_Space4X_Smoke` or chosen capture scene).
- Ensure `Space4XSpectatorRenderBridgeConfigAuthoring` exists and `Enabled=true`.
- Confirm HUD toggle works (`H`).
- Confirm camera controls respond (`F`, `G`, `C`).

## Scenario Order
- Run `space4x_capital_20_vs_20_supergreen` first.
- Run `space4x_capital_100_vs_100_proper` second.

## Per-Scenario Checklist
- Start scenario and let it prewarm.
- Wait 3-5 seconds after load/settle before triggering cinematic.
- Trigger cinematic (`C`) after engagement is readable.
- Keep HUD on for at least one truth take.
- Record 30-60s clip.

## If FPS Hitches
- Initial loading/prewarm hitch is acceptable.
- Do not start cinematic during hitch window.
- Wait 3-5 seconds, then begin take.

## Clip Naming Convention
- `YYYYMMDD_<scenario>_takeNN`
- Examples:
- `20260215_space4x_capital_20_vs_20_supergreen_take01`
- `20260215_space4x_capital_100_vs_100_proper_take01`
