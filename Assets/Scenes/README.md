# Space4x Scenes

This folder contains all Unity scenes for the Space4x project.

## Structure

- **Demo/** - Space4x-specific demo scenes (active)
- **Hybrid/** - Archived hybrid showcase scenes (see ARCHIVED_README.md)

## Notes

- Scenes are independent of Godgame project
- PureDOTS package provides shared DOTS framework
- All assets (prefabs, configs) should be local to this project
- Demo scenes demonstrate Space4x mining/carrier mechanics
