# Legacy Space4X Rendering Stack

This folder stores the pre-PureDOTS rendering bridge (RenderKey authoring, catalog bakers, debug sandboxes, etc.).
It lives outside `Assets/` so Unity ignores it. Use the PureDOTS `RenderPresentationCatalogDefinition`,
`RenderPresentationCatalogAuthoring`, and the semantic/theme/variant/presenter systems for the current pipeline.
